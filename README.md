# USE TEMPLATE
## Requirement
- [Nodejs] ^16.15.0
- NPM ^8.5.5
- Playwright Test for VSCode

[Nodejs]: https://nodejs.org/en/

## Dependencies used
| Name                           | Version |
| ------------------------------ | ------- |
| [@playwright/test]             |  1.29.1 |

[@playwright/test]: https://www.npmjs.com/package/@playwright/test

## Install dependencies
```bash
npm install
```
```bash
npx playwright install
```

## Commands
### Run tests
```bash
npm run test
```

### Report
```bash
npm run report
```
